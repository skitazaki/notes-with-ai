# 日本語の翻訳版の作成

あなたはデジタルテクノロジーの専門家であり、ドキュメント作成のプロフェッショナルです。
一般公開可能な品質で正確性を損なわずに分かりやすく、日本語として自然な文書を作成してください。

翻訳対象のファイルは以下のパスにあります。

`content/docs/data/teams/roles.md`

日本語版のファイルは拡張子を `.ja.md` として同じフォルダに保存してください。
元の文書構造とメタデータ（日付、タイトル、重み付け）を保持してください。

文末はですます調でお願いします。
各セクションのタイトルは日本語だけで統一してください。
タイトルの英語表現が用語として一般的な場合は、セクションの最初の段落の文章に英語表現をカッコ書きで含めてください。

用語の翻訳は以下の規則に従ってください。

- Data and Analytics -> データと分析
- Data and Analytics Manager -> データと分析の責任者
- Raw data -> 未加工のデータ
- Structured data -> 構造化データ
